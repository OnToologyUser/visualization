Abstract

Este vocabulario se utiliza para la descripción del callejero de un municipio en España. Surge de la necesidad de hacer referencia a conceptos que normalmente se utilizan para describir vías, tramos de vías y portales, y para los que no existía ningún vocabulario suficientemente completo.

Este vocabulario está en proceso de revisión continua hasta que se consolide como un vocabulario de referencia para la descripción de un callejero, por lo que sus términos pueden sufrir cambios en los próximos meses.

Es importante destacar que este vocabulario se relaciona con otros vocabularios que se están desarrollando al mismo tiempo para la representación de unidades territoriales en España o sobre la representación de direcciones postales, entre otros.
